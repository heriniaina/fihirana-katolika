• Fihirana Dera p. 51

Hobio I Iaveh, ry tany tontolo, )
Fa mandrakizay ny fitiavany.   ) (indroa)
 
1 Tompoy amim-po falifaly i Iaveh, )
 Ary hatòny amin-karavoana.          ) (indroa)
 
2 Fantaro fa izy no Andriamanitra, )
 Azy isika 'zay harin-tanany.           ) (indroa)
 
3 Ry vahoaka sy ondry fiandriny, )
 Hirao izy eo anivon'ny kianjany. ) (indroa)
 
4 Derao izy, ny anarany ankalazao: )
 Mandrakizay ny fitiavany.              ) (indroa)
 
5 Dera sy laza mandrakizay anie      )
 Ho anao, ry Tompo Andriamanitra. ) (indroa)
