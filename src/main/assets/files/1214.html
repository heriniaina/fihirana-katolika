Mino ny Teninao 'zahay, ry Tompo ô, )
Izany no mahavelona,                        ) (indroa)
Mahavelona mandrakizay.                  )
 
1 Tsy ny hanina ihany no ivelomanay,             )
 Fa ny Teninao, ry Ray, 'zay hany antenainay. ) (indroa)
 
2 Ianao no fiainanay sy fananganana, )
 Ka izay mino anao dia ho velona.      ) (indroa)
 
3 Ho levona tokoa izao tontolo izao,               )
 Fa ny Teninao, ry Tompo, no haharitra doria. ) (indroa)
 
4 Intý aho, Tompo ô, efa vonona tokoa        )
 Hihaino ny Teninao sy handray ny anatrao. ) (indroa)
