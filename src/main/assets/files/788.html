1 Rainay ao an-danitra ao, hohamasinina anie ianao:
 Hotiavina sy hajaina e,     hohamasinina e ny anaranao,
 Ianao no Ray ô,                         >>                    >>
 Ianao no Tompo,                       >>                    >>
 Ianao no Masina e,                    >>                    >>
 Ry Avo indrindra ô,                   >>                    >>
 Zanaharinay ô,                         >>                    >>
 Etý an-tany e,                          >>                    >>
                                         ho tahaka ny ao an-danitra!
 
2 Rainay ao an-danitra ao, fitiavana anie ianao:
 Deraina, ampanjakana e, hampanjakaina e ny fitiavanao,
 Amin'ny tokantranao ô,              >>                   >>
 Amin'ny havan-tiana,                 >>                   >>
 Amin'ny fahavalo,                      >>                   >>
 Amin'ireo madinika e,                >>                   >>
 Amim-piaraha-monina e,           >>                   >>
 Etý an-tany e,                           >>                   >>
                                         ho tahaka ny ao an-danitra!
 
3 Rainay ao an-danitra ao, izao no sitrakao:
 Ny hinoana an'i Jesoa e, hotanterahina e ny sitrakao,
 Harahinay e,                              >>                  >>
 Hiainanay e,                              >>                  >>
 Hambaranay e,                          >>                  >>
 Ho iray ihany e,                         >>                  >>
 Ny zanakao e,                            >>                  >>
 Etý an-tany e,                            >>                  >>
                                         ho tahaka ny ao an-danitra!
 
4 Rainay ao an-danitra ao, misaotra anao 'zahay:
 Tahínao isan'andro,          misaotra anao e ny zanakao,
 Velominao e,                              >>                 >>
 Avela heloka e,                          >>                 >>
 Mba hifamela,                            >>                 >>
 Arovanao e,                               >>                 >>
 Ka velona aina,                          >>                 >>
 Etý an-tany e,                            >>                 >>
                                         ho tahaka ny ao an-danitra!
