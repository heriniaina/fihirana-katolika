• Fihirana Hasina p. 95

1 Mahatsiaro sambatra aho, raha eo anilanao, )
 Mahatsiaro sambatra aho, Tompo ô.               ) (indroa)

Sambatra izay matoky an'Iaveh,
Manokan-tena ho azy mandrakizay;
Sambatra izay matoky an'Iaveh,
Feno haravoana ny fony endrey.

2 Ny vola sy ny harena, tsy misy raha jerena  )
 Raha tsy eo anilanao, Tompo ô.                    ) (indroa)

3 Ny dina izay nekena, dia ankoso-bolamena, )
 Raha tsy eo anilanao, Tompo ô.                    ) (indroa)

4 Ny asa izay atao, dia asa very maina, )
 Raha tsy eo anilanao, Tompo ô.           ) (indroa)

5 Ny koràna rehetra, dia hita misy fetra, )
 Raha tsy eo anilanao, Tompo ô.             ) (indroa)
