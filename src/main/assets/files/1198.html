Tompo ô, hitako maso
'Lay famonjena avy aminao
Nomaninao ho an'ny vahoaka )
Eran'izao tontolo izao.           ) (indroa) [Lk. 2, 29-32]
 
1 Azonao alefa amim-piadanana amin'izay )
 Ny mpanomponao araka ny teninao.         ) (indroa)

2 'Lay hazavana natao hanilo ny firenena, )
 Voninahitray vahoaka voafidinao.            ) (indroa)
 
3 Dia ilay Mesia Zanakao ka nirahinao           )
 Mba hitondra anay hiditra ao an-danitra ao. ) (indroa)
