• Fihirana Hasina p. 193
Faly izahay, ry Tompo,               )
Satria nasandratrao eo anilanao. ) (indroa)
 
Mandrakizay anie - no aoka hiderana
Mandrakizay anie - ny indrafon'ny Tompo
Mandrakizay anie - amintsika vahoaka maminy.
 
Any an-danitra any no miorina ny fahamarinana.
Tompo ô, 'zahay olonao nasandratrao any aminao:
Ho anao anie ny haja, fisaorana.
 
Tompo Andriamanitra ô, navotanao 'zahay,
Indro mitory ny hafaliana, miventy ny hira manao hoe:
 
Mandrakizay anie - no aoka hiderana
Mandrakizay anie - ny indrafon'ny Tompo
Mandrakizay anie - amintsika vahoaka maminy.
