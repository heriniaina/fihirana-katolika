Mijanòna re, ry Tompo ô, (mijanòna e)
Mijanòna hiaraka aminay, (aminay e)
Aza ilaozanao izahay:
Efa hariva anie 'zany andro. (indroa)

1 Fa nahoana ianao no handalo? )
 Tranonao anie ny tranonay.      ) (indroa)

2 Fa ny teninao manafàna )
 Ny fonay sy ny fanahinay. ) (indroa)

3 Eto akaikinao izahay,           )
 Sambatra toa efa an-danitra. ) (indroa)

4 Mba honenanao izahay, )
 Mifamela hadisoana.       ) (indroa)

5 Ny finoanay matimaty     )
 Mba soloy finoana velona. ) (indroa)

6 Herinay ianao, tokinay,  )
 Mba ho tody soa ny alina. ) (indroa)
