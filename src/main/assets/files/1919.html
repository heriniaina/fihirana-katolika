Mba raiso re aho, ry Tompo ô
Hanolo-tena, hanompo Anao
Hanatontosa ny hafatrao
Hampanjaka ny fitiavanao (2)

1.- Ianao no Mpahary (Eny Tompo ô !)
Teny tonga nofo (Eny Tompo ô !)
Iraho aho, hizara fahasoavana (Raiso re ‘ty ‘zaho)
Ianao no antenaina (Eny Tompo ô !)
Hitondra fanavotana (Eny Tompo ô !)
Iraho aho, hitory famonjena (Raiso re ‘ty ‘zaho)

2.- Ianao no Lalana (Eny Tompo ô !)
Sy fahamarinana (Eny Tompo ô !)
Ireto vahoaka, mba ho any aminao (Raiso re ‘ty ‘zaho)
Ianao no Fiainana (Eny Tompo ô !)
Mpiandry Ondry tsara (Eny Tompo ô !)
Mampivondrona ny olona, ho eo aminao (Raiso re ‘ty ‘zaho)

3.- Ianao no mpanompo (Eny Tompo ô !)
Irak’izy Ray (Eny Tompo ô !)
Ampianaro aho mba hanaiky ho mpanompo (Raiso re ‘ty ‘zaho)
Ianao ilay mahantra (Eny Tompo ô !)
Feno hasambarana (Eny Tompo ô !)
Ampio aho hanahaka anao (Raiso re ‘ty ‘zaho)

4.- Vonjin’ireo ory (Eny Tompo ô !)
Andrin’ny madinika (Eny Tompo ô !)
Ampianaro aho hitia ireo tsy tiana (Raiso re ‘ty ‘zaho)
Fonao mangoraka (Eny Tompo ô !)
Mitory famelana (Eny Tompo ô !)
Tariho aho hitory fibebahana (Raiso re ‘ty ‘zaho)
