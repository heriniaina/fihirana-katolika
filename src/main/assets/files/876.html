• Fihirana Hasina p. 98

1 Mitsangàna, ajoroy ny faneva re, )
 Ry tanoran'i Jerosalema!               )
 Faingàna areheto ny jiro e            ) (indroa)
 Hanazava ny lala-mankany!          )

Faingàna, faingàna, faingàna, )
Faingàna, ry namako ô!          )
Andao e, andao e                   ) (indroa)
Hanatona ny Tomponao!        )

2 Ny dimy dia hendry, satria re )
 Efa nanomana solika soa:        )
 Tanteraka sy lavorary e           ) (indroa)
 Ny lalana izorany hatrany!       )

3 Ny dimy dia adala, satria re    )
 Tsy nanomana solika soa:         )
 Nony tonga ilay fampakarana e ) (indroa)
 Dia voahidy ny lala-mankany!   )
