Mifalia, ry feno hasoavana ô, (indroa)
Omban'Andriamanitra fitahiana ianao e,
Nosoavina noho ny vehivavy rehetra,
Ary nosoavina ilay haterakao. (indroa)
 
1 Aza matahotra moa ianao, ry Maria ô, )
 Fa notiavin'Andriamanitra ianao e,        )
 Ka hiteraka zazalahy e, hoy ny Tompo,  ) (indroa)
 I Jesoa no ataovy anarany.                    )
 
2 Ho lehibe sy hanan-daza an'izy e,  )
 Zanaka tokan'ilay Avo indrindra e,   )
 Ka ny razany fantatrao e: i Davida,  ) (indroa)
 Ho mandrakizay ny fanjakany.         )
 
3 Matokisa hono ianao, Maria ô,              )
 Ny Fanahy Masina no hambomba anao e,)
 Ka ny zaza haterakao e dia ny Masina:    ) (indroa)
 Izy no Zanak'Andriamanitra.                   )
 
4 Inty aho ankizivavin'ny Tompo ô,               )
 Aoka ho tanteraka izany teninao e;             )
 'Ndray mientana ny fanahiko e, mankalaza, ) (indroa)
 Faly midera ny Tompontsika.                      )
 
5 Ary ny Teny indro tonga nofo e,   )
 Dia niara-nonina tamintsika izy e.  )
 Voninahitra ho an'Andriamanitra e, ) (indroa)
 any an-danitra,                              )
 Ary fiadanana etý an-tany.             )
