Fiv: Kintan'ny ranomasina,
       Masina Maria reninay e, ( mba henoy e )
       O ry kintana arahina, matoky anao izahay.

1. Fa ny fiainanay toa ranomasi-manonja
    Ny fakam-panahy mila handrendrika.
    Matoky anao e. ( in-2 ) 

2. Ry Kintana iry mamirapiratra ambony
    Toa ilay finoana niainanao, Maria.
     Matoky anao e. ( in-2 )

3. Ka na maizim-be 'zany fiainana etoana
    Ny fanantenana: ianao, ry Kintana ô
    Matoky anao e. ( in-2 )

4. Ny fitiavanao, kintana ambony madio
    No banjininay e manara-dia anao.
    Matoky anao e. ( in-2 )

5. Mba ho tody soa ireto zandrin'i Jesoa,
    Ento amin'ny Ray e, tariho lalana.
    Matoky anao e. ( in-2 )
