• Fihirana Hasina p. 132

Andriamanitra ô, Andriamanitra ô,
Vonjeo (ô vonjeo) vonjeo (ô vonjeo),
Vonjeo, vonjeo ny taninay.

1 Ny lohasaha sy ny havoana,           )
 Ny tendrombohitra izay manoloana ) (indroa)
 Vonjeo, vonjeo, ry Tompo ô.          )

2 Ireo nafafy sy izay nambolena,             )
 Ianao, ry Tompo, no tena andrandraina, ) (indroa)
 Vonjeo, vonjeo, ry Tompo ô.                  )

3 Ireo kamboty 'zay miferinaina      )
 Sy ireo mahantra tsy tambo isaina  ) (indroa)
 Vonjeo, vonjeo, ry Tompo ô.          )
